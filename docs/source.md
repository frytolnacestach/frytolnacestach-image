# Umístění obrázků
Pro obrázky je výchozí adresář ```storage```, kde se nachází již jednotlivé adresáře podle použití obrázku v aplikaci.

## Struktůra adresáře ```storage```
```
├── _default
├── _email
├── articles
├── brands
├── chains
├── events
├── fauna
├── flora
├── foods
├── gallery
├── main
├── maps
├── og
├── travel_dictionaries
├── users
├── videos
├── wall_sockets
└── world
    ├── cities
    ├── continents
    ├── regions
    ├── spots
    └── states
```
